# {{house_name}} Rules

Rules define constraints agents must honor while operating in this house.

## Runtime

- Local-first means OpenResponses through `/bus` before any deployed inference path.
- Browser keys are not required for local operation.
- Inference providers are deployment-testing adapters, not the primary runtime.

## Repository Operations

- Draft plans and action items before making changes.
- Preserve product truth from `README.md`, `AGENTS.md`, package scripts, and workflows.
- Treat missing license, stale workflows, and absent tests as adoption risk.

## Evidence

- Every recommendation should include at least one repo fact, inspection finding, action item, or extraction signal.
