# MLX/OpenResponses Policy

OpenResponses is the runtime contract. Local mode is the primary path and should use `/bus` to reach the MLX-backed gateway at `http://127.0.0.1:8090`.

Inference providers are allowed only as settings-controlled deployed-testing adapters. They should not replace local MLX as the default target, and browser keys should never be required for local operation.

Active Vega Lab mission targets are Codex, Claude, and local MLX. Jules is historical template vocabulary and is not an active target.
