# Tech Stack Defaults

- Frontend: Vite, React, TypeScript, existing Vega Lab design system.
- Data: JSON snapshots mirrored in `data/` and `public/`.
- Tools: MCP/OpenResponses tool contracts from `src/mcp-server/index.js`.
- Runtime: local MLX through `/bus`, proxied to `http://127.0.0.1:8090`.
- Verification: lint, data tests, MCP tests, and build.
